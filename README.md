# Chat Room
A sample work with Node sockets for implement a simple messaging text chat.

### Tech

Dependencies

* [AngularJS]
* [NodeJS]
* [Express]
* [Gulp]
* [Socket.io]

### Installation

Configure Mongo's connection in server/models/ js files


```sh
$ git clone https://github.com/JoseGN/team_chat
$ cd team_chat
$ npm i -d
$ bower install
$ node server.js
```
